复习 数组 1)什么是数组 就是一个能存储多个相同数据类型数据的变量 2)数组的概念 2.1)一个变量可以引用一个数组,称为数组名称 2.2)一个数组中的数据称为数组元素 2.3)每个元素都有一个编号,叫数组下标(从0开始) 2.4)每个数组都有保存数据的上限,称为数组长度 2.5)数组中的数据必须是相同的数据类型 2.6)数组的长度是不能修改的 3)数组的使用 3.1)数组的声明 int[] nums;//推荐 int nums[];//不推荐 3.2)数组的初始化 nums=new int[长度]; 合二为一 int[] nums=new int[5]; 3.3)数组的赋值 直接赋值 前三步合并 int[] nums={5,7,9,1,6,3,4}; int[] nums; nums=new int[]{8,1,6,4,7,5,9}; 3.4)数组的使用 要想使用数组中的元素 数组名[<下标>] 4)数组的复制 4.1)System.arraycopy(1,2,3,4,5); 1.源数组 2.源数组复制起始位置 3.目标数组 4.目标数组复制起始位置 5.复制的元素个数 4.2)Arrays.copyOf(1,2) 1.源数组 2.复制长度 5)数组的排序 5.1)Arrays.sort(<要排序的数组>); 自动排序,结果是升序的 5.2)升序输出和降序输出 笔记 方法 1)什么是方法 java程序中,处理一个功能或业务的代码块 2)为什么要用方法 2.1)将处理不同业务的代码 从程序入口中分离出来,降低代码复杂度 2.2)每个方法专心的编写自己的业务 不和其他业务干扰 2.3)各种方法在main方法中调用 方便团队开发 2.4)方法可以编写一次,调用多次 3)方法的定义 3.1)无参数无返回值的方法 在public class XXX的{}中 public static void 方法名(){ //方法代码(方法体) } 3.2)无参数有返回值的方法 public static <返回值类型> 方法名(){ //方法体 return <返回结果>; } 返回值类型:这个方法运算结果的数据类型 return关键字 1.返回方法的结果 结果的数据类型必须匹配方法的声明 有返回值的方法必须通过return返回 2.运行到return关键字时,方法直接终止 无返回值的方法也可以使用return 但是只是终止方法的含义 3.3)有参数无返回值的方法 public static void 方法名(<参数列表>){ //方法体 } 有参数的方法,需要在方法定义时编写参数列表 参数列表的格式 方法名(参数类型1 参数,参数类型2 参数 ,...) 方法定义时声明的参数,称之为 形式参数(形参) 在方法中把形参看做是已经被赋值的变量 3.4)有参数有返回值的方法 public static <返回值类型> 方法名(<参数列表>){ // 方法体 return <返回的结果>; } 结合了有返回值和有参数方法的注意事项 4)方法的调用 4.1)无参数无返回值的方法 在方法中: 方法名(); 4.2)无参数有返回值的方法 在方法中: 变量=方法名(); 这个变量的类型必须和返回值类型匹配 4.3)有参数无返回值的方法 在方法中: 方法名(<参数列表>); 带参数的方法,需要在调用时传入参数 参数列表的格式 方法名(参数1,参数2,.....) 调用方法时,参数列表称之为 实际参数(实参) 传入的参数的类型\个数\顺序 必须和方法定义时的匹配 4.4)有参数有返回值的方法 变量=方法名(<参数列表>); 作业 方法 1)定义一个无参数无返回值的方法 该方法中输出今天的日期("2019年7月8日") 在main方法中调用该方法,最终显示在控制台 2)定义一个计算圆形面积的方法 输出圆形面积 并调用 3)定义有参数的方法add 参数为int a和int b 方法中输出a+b的和 并在main方法中调用add运行 4)定义个方法doAnySum(int s,int e) 计算输出从s开始累加到e的结果 在main方法中调用 计算50~100累加和 80~200累加和 100~150累加和 草稿 public static void abc(int a,int b){ System.out.println(a+b); } main(){ abc(5,9); int x=10; int y=20; abc(x,y); }
